χirp Report Search

All generated reports in χirp are saved locally as PDF document in the reports folder and additionally in the existing database as backup. To find an older report in the database use the report search dialog. The dialog can be invoked using the menu Reports -> Report search, or the corresponding menu entry in the tray icon context menu.

You can search reports by specifying a time interval and choosing a robot fromm the drop-down menu. It is not nessessary to specify the time or to choose a robot. If you only specify the time interval, reports for every robot int he database are found. If you do not specify a time interval and choose a robot, all reports for this robot are found. Finally if you do not choose a robot and do not specify a time interval all reoprts for all robots are found.
