χirp Profile files

χirp is based on profiles that describe an environment. A profile can contain several different robots. The profiles are saved in *.pro files located in the conf/profiles folder. Thoses files are plain text files structured using XML. A profile must at least specify a name and one robot. The profile can additionally contain external tools the can be started using the generated menu entrys.

Use the profile editor to edit or create new profile files. The external tools can be added/edited using the preferences dialog. Those tools can be invoked using the generated menu entry unter External tools
