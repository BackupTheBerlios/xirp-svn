χirp Recordings

χirp offers the capability to record the values of sensors sent by the robot. To use this feature check the Enable observed recording mode option in the record panel. If this option is enabled you may add one ore more sensor keys to the list of observed keys, visible on the right hand side.



To start the recording, you have to be connected to a robot. In that case, the Start recording button becomes active as seen below.



When the recording was started, the Start recording button turns into the Stop recording button.



When you have finished the recording press the Stop recording button. An input dialog opens and asks for a name and description for the recording. After you filled and closed the input dialog the record is saved to the database.
