χirp Charts

In χirp it is possible to generate several different chart-types. The data on which the charts are generated upon is gathered by the recording feature of χirp, or is directly plotted into a chart by the live-chart feature. Read more about this topics in the corresponding chapters. This chapter deals with the generation of charts from existing records.

Charts from existing records are generated using the Create chart dialog. The dialog is available under the menu entry Extras -> Charts.



There are four possible chart types available. You can choose the from the chart type drop-down menu.

  • Time chart
    This type shows the values as line chart.

  • Pie chart
    This type shows the occurences of keys in the time interval.

  • Bar chart
    This type chart shows the values als bars.

  • Scatter chart
    This type shows the values as points.





Before you can choose a record you have to select one of the robots. This can be done from the robot name drop-down menu.



After that you can choose one or more records for this robot, using the record lookup dialog. If you want only one record to be the data for the chart leave Compare multiple records unchecked. But if you want up to four records to be compared check Compare multiple records. If this flag is set you can select more than one record from the record lookup dialog.



The record lookup dialog shows you all available recordings for the selected robot.



One record

After you have chosen a record you can see the observed keys of this record in the list on the right hand side. You must select at least one of the observed keys. (Learn more about observed keys in the recordings chapter.) In the middle of the dialog you find the time interval area. You can choose the interval to look at by moving the two time separators of the time interval widget.



Before generating the chart you have the possibility to activate some options. You can choose to have a chart generated with relative time values. Use all keys selects all availabe observed keys in the record. Show threshold activates the threshold input field, where you can specify the value where a threshold line is drawn in the chart. The Create 3D chart option is not availabe for all chart types. The last four options are export options, if enabled the chart is exported automatically while after creation. The exports are saved under charts/<robot name>/<export format extension>.



After clicking Generate chart the chart is openend in the generated chart dialog and may look like the one seen below. The chart dialog has several options in its context menu and the ability to zoom into the chart by selecting an area with the mouse.



Multiple records



If you choose to compare up to four records you have to check Compare multipe records. Then you select up to four records form with the record lookup dialog. On the right hand side you can see the observed keys which have all the selected records in common. Also the time interval area has an additional option, the Time interval value in milliseconds. This value is the amount of milliseconds to look at in the selected records. The interval can only be as large as the shortest record. The sliders in the time interval widget now only can be moved togerther int the chosen interval.
