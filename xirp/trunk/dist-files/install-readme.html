χirp @VERSION@ overview

The χirp 2 Framework

χirp stands for eXtendable Interface for Robotic Purposes, and aims to provide a feature rich platform for rapid development of graphical user interfaces for robot control.

The delivered robot profiles

You will may install a profile for a robot which may be used for testing: The TesterBot

The delivered plugins

There are some plugins which come with this χirp installation.
Most of them are default plugins for visualization of the sensors of the TesterBot.

Sourcecode

It's possbile to install the zipped sourcecode of χirp.

Documentation

χirp provides a user guide which describes which functionality the application provides and how it is used. There's also a developer guide which provides information and examples for plugin development.
